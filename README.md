# 🌌 SYNAPSE CORE
### Next-generation Intelligent API Infrastructure

**Synapse Core** is an ultra-premium AI Data Intelligence Platform designed for high-performance data operations and real-time monitoring.

![SYNAPSE CORE UI](public/assets/preview.png) *(Note: Add your own preview screenshot here)*

## 🚀 Features
- **Cinematic Reveal**: High-tech "merged line" title reveal on initialization.
- **System Boot Preloader**: Immersive tactical loading sequence with live system logs.
- **Neural Activity Graph**: Real-time canvas-based health monitoring.
- **Live Terminal Stream**: Scrolling console logs for all REST API operations.
- **Tactical CRUD Dashboard**: Manage neural nodes with holographic interactions.
- **Enterprise Monitoring**: Live metrics for CPU Load, Memory, and Active Tasks.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+).
- **Backend**: Node.js, Express.js.
- **Animations**: CSS Keyframes, Canvas API.
- **Styling**: Vanilla CSS with futuristic "Deep Space" palette.

## 📦 Local Setup
1. **Clone the repository**:
   ```bash
   git clone https://github.com/ManoharK-777/synapse-core.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the server**:
   ```bash
   node server.js
   ```
4. **Access the platform**: Open `http://localhost:5000` in your browser.

## 🌐 Deployment
### Cloudflare Pages (Frontend)
1. Build your project or use the `public/` folder directly.
2. Connect this GitHub repository to Cloudflare Pages.
3. Set the build output directory to `public`.

### Render (Backend)
1. Create a new "Web Service" on Render.
2. Connect this repository.
3. Use `node server.js` as the start command.
